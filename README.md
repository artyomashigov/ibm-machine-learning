# IBM Machine Learning Course

Final assignment work for IBM's Machine Learning course.

## Project Goal

This repository contains the final course notebook for practicing the end-to-end machine learning workflow: loading data, preparing features, training models, evaluating results, and explaining the modeling process.

## Files

- `ML0101EN_SkillUp_FinalAssignment.ipynb` - final assignment notebook.

## Workflow

1. Load the assignment dataset.
2. Inspect and prepare the data for modeling.
3. Train machine learning models.
4. Evaluate model performance.
5. Summarize the results and conclusions.

## Tools

Python, pandas, scikit-learn, and Jupyter Notebook.
