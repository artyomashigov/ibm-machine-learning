# IBM Machine Learning Course

Final assignment notebook for IBM's Machine Learning course.

## Project Overview

This repository contains the final course assignment notebook. The notebook follows a complete supervised machine learning workflow: importing a dataset, preprocessing features, splitting data, training models, evaluating predictions, and comparing model performance.

## Notebook Structure

- Assignment instructions and dataset overview
- Required library imports
- Dataset loading
- Data preprocessing
- One-hot encoding for categorical variables
- Train/test split
- Linear Regression model
- Prediction and metric calculation
- Additional supervised learning models from the course workflow
- Model comparison and interpretation

## Methods

The notebook uses scikit-learn to build and evaluate machine learning models. It includes train/test splitting, feature preprocessing, prediction generation, and metric-based evaluation.

Typical evaluation metrics in the assignment workflow include regression or classification metrics such as error measures, accuracy-style scores, and model comparison outputs depending on the section.
